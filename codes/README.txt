- main_undet_coeffs.m 
  Matlab script that solves the stochastic neoclassical growth model
  using log-linearization and method of undetermined coefficients.

- main_with_dynare.m 
  Matlab script that solves the stochastic neoclassical growth model
  using Dynare. It needs the file stochastic_growth_dynare.mod
  
- main_with_dynare2.m 
  Matlab script that solves the stochastic neoclassical growth model
  using Dynare. It needs the file stochastic_growth_dynare2.mod
  With respect to main_with_dynare2.m, it is more advanced.
